## Tutorial

Faça o download .zip (unzip ele em seguida),
acesse a pasta ou via Terminal (cmd, git, etc...)

Para instalar use:

`npm install` ou `npm i` (obs.: dentro da pasta do arquivo)

E agora só rodar o código!

Use: `node .`
Arquivo padrão para iniciar é `index.js`
